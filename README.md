# project_pzad
---
Данный проект реализован на Python версии 3.10.0
Для установки окружения для корректной работы ноутбука необходимо установить 
набор библиотек, указанный в файле requirements.txt, и сделать следующее в терминале:
```bash
pip install -r requirements.txt
```

Сама модель и датасет загружаются локально, так как они очень тяжеловесны. В силу того, что гитхаб не принимает файлы такого размера, оставим здесь ссылку на яндекс диск с сответствующими файлами: https://disk.360.yandex.ru/d/VZFMZCucuzH0fQ
