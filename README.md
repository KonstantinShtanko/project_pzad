## Executive Summary

Разработка специализированной нейросетевой модели для автоматической генерации мемов по текстовому описанию. В основе решения лежит дообучение (fine-tuning) передовой открытой диффузионной модели AltDiffusion на профильном датасете `d0rj/text_meme`, содержащем пары «описание — изображение мема».

**Ключевая цель** — создать инструмент, который по короткому текстовому запросу пользователя будет генерировать уникальное, релевантное и стилистически соответствующее жанру мема изображение.

### Техническая реализация:
- Для эффективной адаптации модели под специфику мемов будет применена **LoRA (Low-Rank Adaptation)**, что позволит достичь требуемого качества при оптимальных вычислительных затратах.
- Качество генерации будет оцениваться с помощью **CLIP-score** и экспертной оценкой с участием людей (human evaluation).

### Ожидаемый результат:
- Работоспособная дообученная модель, способная генерировать изображения в формате мемов.
- Интегрированный с моделью Telegram-бот для демонстрации функциональности и сбора обратной связи.
- Подробный отчёт с результатами экспериментов, метриками и выводами.

## Описание проекта

В современной цифровой среде мемы стали универсальным языком коммуникации, активно используемым в маркетинге, SMM, внутренних коммуникациях компаний и развлекательных сервисах. Однако создание качественного и оригинального мема остаётся трудоёмким процессом.

Существующие нейросетевые генераторы изображений (Midjourney, DALL-E, Kandinsky и др.) не специализированы на задаче создания мемов. 

**Наиболее распространённые проблемы:**
- Часто генерируют изображения, не соответствующие «мемному» стилю (например, слишком фотореалистичные или, наоборот, абстрактные).
- Требуют сложного, многоступенчатого промптинга для получения нужного результата.
- Не гарантируют релевантность сгенерированной картинки задуманному юмористическому посылу.

Спрос на автоматизацию творческих процессов растёт. Появление доступных и качественных генеративных моделей открывает возможность для создания узкоспециализированных инструментов.

Для маркетинга и SMM это возможность быстро создавать большой объём контента для социальных сетей, адаптированного под текущие инфоповоды и целевую аудиторию. Это повышает вовлечённость и снижает затраты на контент-мейкеров.

### Преимущества:
- Будучи дообученной на реальных мемах, модель сможет улавливать тонкие стилистические особенности жанра (типичные ракурсы, выражения лиц, цветовые схемы, шаблоны).
- Интеграция с Telegram-ботом делает решение доступным для массового пользователя без специальных навыков.

## Исходные данные

**Датасет:** `d0rj/text_meme` (https://huggingface.co/datasets/d0rj/text_meme)

**Структура:** пары «текстовое описание — изображение мема». Язык описаний — русский.  
**Объём:** 6239 строк.

**Пример:**
> текст сверху: "когда решил на пару секунд закрыть глаза после будильника".  
> ниже кадр из фильма "Интерстеллар", где изображён человек в скафандре за управлением космического корабля — Купер.  
> текст ниже: "этот маленький манёвр будет стоить нам 51 год".

## Базовая модель AltDiffusion

(https://huggingface.co/black-forest-labs/FLUX.1-dev)

**Характеристики:**
- Мультиязычная диффузионная модель, поддерживающая 18 языков (включая русский)
- Текстовая архитектура на базе AltCLIP (модифицированный CLIP с кодировщиком XLM-R), прошедшая двухэтапное обучение: выравнивание концепций и улучшение качества
- Открытые веса на Hugging Face, полная интеграция с библиотекой `diffusers` (класс `AltDiffusionPipeline`)

**Дообучение:** LoRA, accelerate  
**Интеграция:** FastAPI, python-telegram-bot

## План работы

### 1. Дообучение модели:
- **Метод:** LoRA (Low-Rank Adaptation). Подбор гиперпараметров.
- **Инструменты:** `diffusers`, `transformers`, `trl` (для удобства обучения), `accelerate`.

### 2. Эксперименты и оценка:
- **Сравнение:** Генерация одной и той же промпт-тестовой выборки базовой моделью и нашей дообученной.
- **Автоматические метрики:**  
  *CLIP-score*: Оцениваем косинусную близость между эмбеддингами текста и изображения для тестовой выборки. Ожидаем рост по сравнению с базой.
- **Оценка с участием людей (Human Evaluation)**

### 3. Интеграция с Telegram-ботом:
**Архитектура:**
- Telegram Bot (на `python-telegram-bot`) принимает сообщение от пользователя.
- Передаёт текст в API-сервис инференса.
- API-сервис (FastAPI) загружает базовую модель AltDiffusion и LoRA-веса, генерирует изображение.
- Возвращает картинку боту, бот отправляет пользователю.
- **Обработка ошибок:** Таймауты, повторные попытки, вежливые сообщения об ошибках.

## Потенциальные сложности

- **Генерация неуместного контента**  
  *Решение* — фильтрация: стоп-лист запрещённых слов в запросах.

- **Отсутствие текста на мемах**  
  *Решение* — возможна интеграция отдельной модели для наложения текста (AnyText/GlyphControl)

- **Быстрое устаревание мемов**  
  *Решение* — регулярное дообучение модели на свежих данных. Для эффективности используется тот же LoRA-подход — добавление новых адаптеров поверх базовой модели без переобучения с нуля.
